# DSA Problems and Notes

A collection of Data Structures and Algorithms problems, solutions, and topic-wise notes.

## Directory Structure

```
├── backtracking/
│   ├── problems/     # Problem solutions
│   └── notes/        # Topic notes
├── graph/
│   ├── codes/        # Graph implementations
│   └── Notes/        # Graph theory notes
└── pattern/          # Pattern problems
```

## Topics

### Backtracking
- Largest Number with K Swaps
- N-Digit Number with Monotonic Digits
- Rat in a Maze

### Graph
- Graph implementations
- Graph algorithms (BFS, DFS, etc.)

### Pattern
- Pattern-based problems
